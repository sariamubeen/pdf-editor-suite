PDF Editor Suite

Client Setup Guide

Installation

1
Double-click INSTALL.bat
2
If asked for admin permission, click Yes
3
When it asks for the server URL, just press Enter to accept the default
4
Wait for "Setup Complete" and press any key to close

That's it. You're ready to edit PDFs.

How to Open a PDF

If PDF Editor Suite is your default PDF app

Just double-click any PDF file. It will open in the browser editor automatically.

If another app opens instead (e.g. Adobe, Edge)

1
Right-click the PDF file
2
Click Open with
3
Select PDF Editor Suite (Browser)

The PDF will upload to the server and open in the browser editor.

Don't see "PDF Editor Suite (Browser)" in the list?
Click "Choose another app" → scroll down → "More apps" → "Look for another app on this PC" → go to C:\Program Files\PDFEditorSuite\ → select open-pdf.bat

What You Can Do

In the editor you can:
  • View and read PDFs
  • Edit text in PDFs
  • Fill in PDF forms
  • Add highlights and annotations
  • Add comments
  • Draw on the PDF
  • Download the edited file
  • Print

Troubleshooting

Browser opens but nothing loads
The server may be down. Contact your IT admin.
PDF doesn't appear in the editor
Open File Explorer, type %TEMP% in the address bar, and open PDFEditorSuite-debug.log to see what went wrong.
"Enter a name for collaboration" popup appears
Type your name and click OK. This only appears once.

Uninstall

Double-click UNINSTALL.bat, or run C:\Program Files\PDFEditorSuite\Uninstall.bat
